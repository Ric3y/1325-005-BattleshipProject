# Byttleship

Text based battleship-like game made using Java 

## How to use

- Clone the project at 
```
git clone https://github.com/Ric3y/1325-005-Byttleship.git
 ```

- Go to the folder directory and compile the files
```
javac -d . *.java
```

- Start the program
```
java BattleshipGame.java
```
